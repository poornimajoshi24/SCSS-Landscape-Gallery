# SCSS Landscape Gallery

A simple interactive landscape gallery built using **SCSS** and **Flexbox**.  
The project showcases hover-based expansion effects with smooth transitions.

## Features
- Built using **SCSS (Sass)** for cleaner and modular styling
- Flexbox layout with dynamic `flex-grow` transitions
- Smooth hover animations using `cubic-bezier`
- Responsive full-screen layout
- Interactive cursor effects

## Technologies Used
- HTML5
- SCSS (Sass)
- CSS Flexbox
-  CSS

## Preview
Hover over the landscape panels to see smooth expansion effects.

---

This project demonstrates how SCSS can be used to write structured, reusable, and maintainable styles for interactive UI components.
